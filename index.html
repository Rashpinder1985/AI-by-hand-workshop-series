Workshop Overview

This interactive series combines paper‑and‑pen math, beginner‑friendly Python coding, and AI research insights using the AI by Hand Deep Learning Math Workbook (India Edition).

  • Activation, Neurons, Batch Processing
  • Hidden Layers • Deep vs. Wide Networks • Softmax
  • Gradients & Backpropagation

Includes

  • Paper‑and‑pen math walkthroughs
  • Simple Python code templates
  • References to foundational AI research
